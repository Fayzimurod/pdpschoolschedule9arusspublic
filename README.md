# 📅 Interactive Schedule Web App

A modern, responsive web application for displaying weekly class schedules with real-time clock and interactive calendar. Built with pure HTML, CSS, and JavaScript.

## ✨ Features

- **Real-time Clock** - Live updating time and date display
- **Weekly Schedule** - Clean, organized timetable for Grade 9A Russian
- **Dual Theme Support** - Toggle between Light and Dark themes
- **Interactive Calendar** - Navigable month calendar highlighting current day
- **Fully Responsive** - Perfect display on all devices
- **Rich UI Effects** - Smooth transitions and hover animations

## 🛠️ Technologies

- **HTML5** - Semantic markup structure
- **CSS3** - Flexbox, Grid, CSS Variables, Animations
- **JavaScript** - ES6+, DOM manipulation, LocalStorage
- **LocalStorage API** - Theme preference persistence

## 🎯 Key Features Explained

### Theme Switcher
- Toggle between light and dark themes
- Theme preference saved in browser's localStorage
- Smooth transition effects between themes

### Real-time Clock
- Updates every second
- Displays current time in HH:MM:SS format
- Shows full date with day name, month, and year

### Interactive Calendar
- Navigate between months using arrow buttons
- Current day highlighted automatically
- Days from other months shown with reduced opacity
- Responsive grid layout

## 📱 Responsive Design

The application is fully responsive with breakpoints:
- **Mobile** (< 400px): Compact layout with smaller text
- **Tablet** (768px - 1024px): Optimized spacing
- **Desktop** (> 1024px): Full layout with larger elements

## 🎨 Customization

Dark theme:
- Background: #1a1a1a
- Text: #e0e0e0
- Accent: seagreen (adjusted opacity)

### Font Family
- Primary: sans-serif
- Fallback: system-ui

## 💡 Usage

- **View Schedule** - The main table shows the weekly timetable
- **Switch Themes** - Click the theme toggle button (LIGHT/DARK)
- **Navigate Calendar** - Use ← and → arrows to change months
- **Check Current Time** - Top-left corner shows real-time clock

## 🔧 Customization

### Modify Schedule
Update the table structure in `index.html` to change classes, times, or days.
